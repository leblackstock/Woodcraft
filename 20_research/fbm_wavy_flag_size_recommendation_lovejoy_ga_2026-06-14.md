## Recommendation

Start with:

- **Primary FBM size:** 24.7 in. long x 13 in. tall x 1.5 in. thick
- **Listing price test:** $100-$125 if finish/photos look premium; be prepared to accept or test $90-$100 if messages are slow
- **Copy angle:** compact handmade wavy flag, real wood grain in the natural stripes, ready-to-hang or hardware included if true

Offer as upgrade/custom:

- **Large/premium size:** 37 in. long x about 19.5-20 in. tall x 1.5 in. thick
- **Listing price test:** $175-$225 locally, depending on star method, finish quality, hardware, and photos

## Practical Note

If only one prototype is being made for the first FBM test, make the **24.7 in.** version. A 37 in. version can stay parked as a later premium upsell after the smaller local FBM test gets message data.
