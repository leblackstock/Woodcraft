# Woodcraft Historical Archive

This folder preserves completed, superseded, deprecated, and retired workspace material without making it part of routine discovery.

Default repository searches exclude this folder through the root .ignore file. Search it only when history is explicitly needed:

    rg --no-ignore "search phrase" 90_archive

## Categories

- deprecated_prompts/ — retired prompt generators with a known active successor.
- retired_content/ — retired content-specific generation artifacts.
- retired_content_records/ — inactive-SKU social records retained only as historical evidence.
- retired_workflows/ — inactive workflow paths retained for decision traceability.
- maintenance_audits/ — dated audit reports.
- session_briefings/ — one daily operational snapshot per America/New_York date.
- weekly_reviews/ — completed weekly-review records.

Use RETIREMENT_LEDGER.md to find the original location, successor, and reason for an archived file. Historical files are preserved; do not treat them as current instructions.
